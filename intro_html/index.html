Tags de texto

HTML, é a linguagem fundamental para a criação de páginas da web. Criada por Tim Berners-Lee no início dos anos 90, o HTML serve para estruturar o conteúdo de uma página da web, permitindo a inclusão de textos, imagens, vídeos, links e outros elementos interativos. Ele é composto por elementos, que são definidos por tags, que delimitam o início e o fim de um elemento específico, e atributos, que fornecem informações adicionais sobre o elemento.
